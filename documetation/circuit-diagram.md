# Circuit Diagram - Obstacle Detection System

## Components List:
1. Arduino Uno R3
2. HC-SR04 Ultrasonic Sensor
3. IR Obstacle Sensor
4. RGB LED Module
5. Active Buzzer
6. Breadboard
7. Jumper Wires (some salvaged from DVD player)

## Connection Map:
- Trig Pin → Arduino Pin 9
- Echo Pin → Arduino Pin 10
- IR Sensor → Analog Pin A0
- RGB LED → Pins 5,6,7
- Buzzer → Pin 8

## Power:
- 5V from Arduino board
- All sensors share common ground
